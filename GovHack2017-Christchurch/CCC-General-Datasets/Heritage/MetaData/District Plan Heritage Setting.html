    Citation:
      Citation Information:
        Title: District Plan Heritage Setting
    Description:
      Abstract:
        This entity describes a provision made in the District Plan to protect the surroundings which contribute to the distinctive cultural and historic value of buildings, places or objects.
        A heritage setting must always have one or more related heritage items. Though they may not necessarily be spatial coincident.

        It is spatially abstracted to an Area.
      Purpose: Management of Rules within the District Plan.
      Supplemental Information:
        Maintaining in SQL database via GeoMedia Only to be maintained within the District Plan processes e.g. Plan Change.

    Detailed Description:
      Entity Type:
        Entity Type Label: vwDistrictPlanHeritageSetting
        Entity Type Definition:
          This view combines data from vvwDistrictPlanHeritageSetting and its related domain tables to ensure meaningful attributes for the end user. It is spatially abstracted to an area.
      Attribute:
        Attribute Label: DistrictPlanHeritageSettingID
        Attribute Definition: System generated unique identifier
      Attribute:
        Attribute Label: HeritageSettingName
        Attribute Definition: The short description of the surroundings of the object as defined in the District Plan Schedule of Significant Historic Heritage Places (e.g.Former Canterbury College).
      Attribute:
        Attribute Label: HeritageSettingStatus
        Attribute Definition: Foreign key to dbo.domDistrictPlanHeritageSettingStatus domain table. Limited list of values to describe the currency of the data record in the CCC GIS system (eg Proposed, Current).
      Attribute:
        Attribute Label: LastEditDate
        Attribute Definition: This attribute provides a record of the datetime when the record was last edited, recorded as a datetime captured by a trigger from getdate()
      Attribute:
        Attribute Label: LastEditUserName
        Attribute Definition: This attribute provides a record of the user when the record was last edited, recorded as a CCITY\Username captured by a trigger from suser_sname()
      Attribute:
        Attribute Label: ScheduleReference
        Attribute Definition: The official numeric identifier of the surroundings of the object as defined in the District Plan Schedule of Significant Historic Heritage Places (e.g. 200, 613).
